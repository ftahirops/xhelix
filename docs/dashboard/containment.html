Containment Ladder

Graduated 9-step response. Each step requires confidence threshold + Soak gate. PanicSwitch rolls back steps 3-9 with one command.

StepActionReversible?
1Log onlyn/a
2Snapshot evidencen/a
3Alert + freeze further escalationyes
4Block credentials at kernel level (credbroker)yes
5Block outbound network destinations (netban)yes
6Quarantine files (chattr +i + move)yes
7Kill all processes for this sourcepartial
8Lock account + force credential rotationyes (via operator)
9Host-wide isolation (cut all egress)yes (via PanicSwitch)
Protect-our-own applies here too
Step 6 (quarantine) cannot target paths in ProtectedSystemPaths regardless of which rule fires. See Protect-Our-Own Backstop.
