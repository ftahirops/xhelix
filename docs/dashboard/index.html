xhelix documentation

Open-source Linux server EDR. Three-plane architecture: endpoint enforces locally, hub analyzes deeply, curator alone changes trust. This site is the canonical operator-and-developer reference.

v2 last updated 2026-05-23 Apache-2.0 Linux ≥ 5.15
What xhelix is, in one sentence
A program that watches what's happening on a Linux server, decides if any of it looks like an attack, and stops the dangerous parts before they cause damage — without trusting the application that's being attacked.
