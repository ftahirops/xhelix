Source Lineage

Every event on the host is attributed to an authenticated entry point via SourceAnchor + CausalSet. Propagates across spawn, file taint, network connect, and ptrace injection.

Core concepts

  • SourceAnchor — minted at every authenticated entry point (sshd accept, PAM session_open, sudo, cron fire, systemd start). Has unique 64-bit ID, parent_anchor_id for pivot chains, persisted to /var/lib/xhelix/source.db.
  • CausalSet — bounded (16) deduplicated set of anchor IDs that contributed to a process. Becomes Ambiguous when ≥2 sources merge — the injection-detection signal.
  • PrimarySource — most-recent contributing anchor on a process. Used for operator display and quick-score path.
  • File taint — bounded LRU map of path → (writer primary, writer CausalSet). Reader inherits writer set on tracked paths.

Propagation rules

EventBehavior
Process spawnChild inherits parent PrimarySource + CausalSet
Identity event with PIDAttributeSource on the live PID — adds anchor to CausalSet, updates PrimarySource
FIM writeFileTaint records (path → writer source)
file_open readFileTaint lookup; MergeCausalSet(reader_pid, writer set)
net_connectTags event with source_anchor_id + source_set_hash + ambiguous flag
ptrace_attachTarget inherits attacker source → Ambiguous flips → injection signal

CLI surface

xhelixctl source list --limit 50
xhelixctl source show 42
xhelixctl source children 42
xhelixctl source count

Visual graph rendering (xhelixctl source 42 graph) is part of T04.
