Multi-Chain Composition

8 parallel chains track different threat surfaces. An incident emerges when ≥2 chains converge on the same (source, asset class, destination class) tuple. Endpoint runs a 5-chain quick-score; hub runs the full 8-chain deep composition.

The 8 chains

  • process_chain · file_read_chain · file_write_chain · secret_access_chain
  • network_egress_chain · identity_session_chain · persistence_chain · privilege_chain

Convergence logic

Each chain is a stream of (source_anchor_id, time, action, target). Convergence = same source_anchor across ≥2 chains within a time window. The 4 interaction bonuses (from v2 verification doc): SecretTouch+NovelOutbound, WebRole+ShellSpawn, ProtectedWriteBurst+EntropyJump, TmpExec+SteadyTLSOutbound.
