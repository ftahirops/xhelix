Data Flow & Storage

Where data lives on the endpoint, where it lives on the hub, and why both are needed (not duplicate).

Endpoint storage

WhatWhereTechWhy local
Recent events (24h ring)/var/lib/xhelix/hot.dbSQLite (modernc.org/sqlite)TUI needs instant query; hub round-trip too slow
Source anchors (30 day retention)/var/lib/xhelix/source.dbSame SQLiteAttribution must survive restarts and offline periods
Forensic chain (signed)/var/lib/xhelix/chain.dbSQLite + Ed25519Authoritative legal evidence; hub gets stream copy only
BRP signed profiles/etc/xhelix/brp/*.signed.jsonJSON + Ed25519Cached so endpoint enforces even if hub is down
Process treeRAMHash map + LRULive state; recreated on restart
File taintRAMLRU map, 24h TTLShort-lived; not worth uploading

Hub storage
