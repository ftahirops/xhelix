Detection Stack

Per-event flow through the three gates: L0 hard-deny, L1 BRP envelope verifier, L2 fast-clear. Most events stop at L1; 1% enter L2 deep composition.

Event flow (verbal)

  1. Sensor emits — eBPF, FIM, identity tailer, capwatch.
  2. EAC admits — dedup + ordering + backpressure.
  3. Pipeline enriches — source_anchor_id stamp, proctree ancestry, BRP envelope tag.
  4. L0 gate — protected-paths / forever-suspicious → BLOCK + freeze + snapshot.
  5. L1 gate — 8 domain LLR scoring → 4-state verdict.
  6. L2 gate — Suspicious → multi-chain composition.
  7. Containment ladder — graduated response 1-9.
  8. Forensic chain — Ed25519 batch sign every event.
